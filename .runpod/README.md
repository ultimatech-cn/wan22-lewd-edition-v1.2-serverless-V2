# RunPod Serverless Template

Generic ComfyUI serverless template for RunPod.

This repository is designed as a reusable starter for client delivery projects:

- Mature `handler.py`
- Network Volume based model strategy
- Config-driven custom node installation
- Config-driven model download manifest

Before publishing, update:

- `.runpod/hub.json`
- `project-config/custom-nodes.txt`
- `project-config/model-manifest.txt`
- Root `README.md`
